    Rewriter

    Rewriter is the top level namespace in a synvert snippet. One rewriter checks if the dependency version matche, and it can contain one or many Instances, which define the behavior what files need to find and what codes need to rewrite.

  addSnippet("https://github.com/xinminlabs/synvert-snippets-javascript/blob/main/lib/javascript/no-useless-constructor.js")
